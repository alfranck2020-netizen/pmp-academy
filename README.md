# PMP Master Academy 🎯

Application de préparation à la certification PMP basée sur le PMBOK 7.

## Fonctionnalités
- 📚 Guide PMBOK 7 complet (12 principes + 8 domaines)
- 🎯 Diagnostic de niveau adaptatif
- 🃏 Flashcards SRS (répétition espacée)
- ⚡ Sprint 15 min (entraînement intensif)
- ⏱️ Simulateur examen style Pearson VUE
- 📊 Progression & Badges
